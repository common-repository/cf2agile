Contact Form to Agile

== Description ==

Contact Form to Agile is a powerful Add-on for Contact Form 7 plugin which extends the standard features of the contact form by adding new fields to interact with Active Collab.

Features plugin.

- Creating new projects in Active Collab.
- Adding budged to the new and existing projects.
- Invitation clients and companies to the new projects.
- Adding clients and companies to the new projects.
- Opportunity to create note with all data created in contact form.

= Installation =

1. Upload the plugin files to the `/wp-content/plugins/cf2agile` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
